## Layout Contract

### Desktop (>=768px)

```
+------------------------------------------------------+
|  [Country] [City] [Firm] [Map]          (ScaleTabs)   |
+------------------------------------------------------+
|                                     [Dimension Toggle] |
|        Composite | M | I | N | D       (right-aligned)|
+------------------------------------------------------+
|                                                        |
|     +-----------+                                      |
|     |           |  Choropleth World Map                 |
|     |  visualMap |  aspect-ratio: 2/1 (desktop)         |
|     |  legend   |  max-width: 100%                      |
|     |  (vertical)|                                      |
|     +-----------+                                      |
|                                                        |
|     [Reset View]  (bottom-left of map container)       |
+------------------------------------------------------+
|                                                        |
|  Country Detail Card (shown after click):              |
|  +--------------------------------------------------+ |
|  | Radar Chart              | Binding Constraint     | |
|  +--------------------------------------------------+ |
|                                                        |
+------------------------------------------------------+
```

- Map container: `width: 100%`, `aspect-ratio: 2/1`, `min-height: 400px`
- VisualMap legend: vertical orientation, positioned `right: 10px`, `bottom: 20px` within ECharts
- Dimension toggle: flex row, right-aligned, `margin-bottom: 16px` above map
- Country detail: same card layout as existing country-detail section
- Reset-view button: absolute positioned, `bottom: 12px`, `left: 12px` inside map container

### Mobile (<768px)

```
+-----------------------------+
| [Country][City][Firm][Map]  |
+-----------------------------+
| Composite | M | I | N | D  |
| (full-width, centered)     |
+-----------------------------+
|                             |
|  Choropleth World Map       |
|  aspect-ratio: 4/3          |
|  min-height: 280px          |
|  touch: pinch-zoom + drag   |
|                             |
|  [Reset View]               |
+-----------------------------+
|  VisualMap (horizontal,     |
|   below map)                |
+-----------------------------+
|                             |
|  Country Detail (stacked)   |
|  Radar Chart (full-width)   |
|  Binding Constraint         |
|                             |
+-----------------------------+
```

- Map container: `width: 100%`, `aspect-ratio: 4/3`, `min-height: 280px`
- VisualMap legend: horizontal orientation, positioned below map as separate element or `bottom: 0` with `orient: 'horizontal'`
- Dimension toggle: full-width, centered, wraps to two rows if needed
- Touch interactions: `roam: true` with pinch-zoom and drag-to-pan
