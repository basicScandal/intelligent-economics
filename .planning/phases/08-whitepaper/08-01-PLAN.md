**Step 2: Update astro.config.mjs**

Add MDX integration with remark-math and rehype-katex plugins. The file currently has only tailwindcss vite plugin. The updated file should be:

```javascript
import { defineConfig } from 'astro/config';
import tailwindcss from '@tailwindcss/vite';
import mdx from '@astrojs/mdx';
import remarkMath from 'remark-math';
import rehypeKatex from 'rehype-katex';

export default defineConfig({
  site: 'https://intelligenteconomics.ai',
  integrations: [
    mdx({
      remarkPlugins: [remarkMath],
      rehypePlugins: [rehypeKatex],
      remarkRehype: { footnoteLabel: 'References' },
    }),
  ],
  vite: {
    plugins: [tailwindcss()],
  },
});
```

CRITICAL: `remarkPlugins` and `rehypePlugins` go INSIDE the `mdx()` integration config, NOT at the top-level `markdown` key. The `remarkRehype: { footnoteLabel: 'References' }` configures remark-gfm (bundled in @astrojs/mdx) to label the footnotes section as "References" (per PAPER-04).

## Introduction

For over eighty years, Gross Domestic Product has served as the world's default measure of economic health. Yet GDP was never designed to measure prosperity — it measures *production*. This paper introduces the MIND framework as a complement to GDP that captures the dimensions most critical to long-term, sustainable prosperity.

## The MIND Framework

MIND stands for **Material**, **Intelligence**, **Network**, and **Diversity** — four dimensions that together capture a more complete picture of economic health than any single metric.

## Mathematical Foundation

The MIND score is computed as a geometric mean of four normalized dimension scores:

$$
P = \left(\frac{M}{100} \cdot \frac{I}{100} \cdot \frac{N}{100} \cdot \frac{D}{100}\right)^{0.25} \times 100
$$

Where each dimension score $D_i$ is normalized to a $[0, 100]$ scale using percentile-capped min-max normalization.

The multiplicative structure is deliberate: unlike an additive index, a zero in any single dimension collapses the entire score. This *zero-floor property* captures the reality that prosperity requires balance — excellence in three dimensions cannot compensate for catastrophic failure in a fourth.

## Conclusion

The MIND framework offers a structured, data-driven alternative to GDP-centric analysis. Further sections will be added in subsequent updates.
