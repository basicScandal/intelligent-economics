**Step 1: Add IntersectionObserver scroll-spy to WhitepaperLayout.astro**

Add an inline `<script>` tag (NOT an island — too trivial for hydration overhead) at the bottom of WhitepaperLayout.astro, before the closing `</BaseLayout>` tag. This script:

1. Selects all `h2` and `h3` elements inside `article`
2. Creates an IntersectionObserver with `rootMargin: '0px 0px -66%'` and `threshold: 1`
3. On intersection, removes `active` class from all `#toc a` links, then adds `active` to the matching link
4. Runs immediately on page load (no DOMContentLoaded wrapper needed — script is at bottom)

```html
<script>
  function initScrollSpy() {
    const setCurrent = (entries) => {
      for (const entry of entries) {
        const id = entry.target.id;
        const link = document.querySelector(`#toc a[href="#${id}"]`);
        if (!link) continue;
        if (entry.isIntersecting) {
          document.querySelectorAll('#toc a').forEach(a => a.classList.remove('active'));
          link.classList.add('active');
        }
      }
    };

    const observer = new IntersectionObserver(setCurrent, {
      rootMargin: '0px 0px -66%',
      threshold: 1,
    });

    document.querySelectorAll('article :is(h2, h3)').forEach(h => observer.observe(h));
  }

  initScrollSpy();
</script>
```

Do NOT use `client:visible` or any island directive. This is a plain Astro `<script>` tag that Astro will bundle and deduplicate.
