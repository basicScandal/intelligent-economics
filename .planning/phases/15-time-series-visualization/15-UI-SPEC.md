## Visual Hierarchy

Primary focal point: the time-series line chart showing MIND score evolution. Secondary: the year controls bar (play/pause + slider). Tertiary: multi-country legend and speed toggle. The year controls bar becomes a shared element visible across Map and Country tabs when a non-latest year is active.

## Interaction States

### Year Slider States

| State | Appearance | Trigger |
|-------|-----------|---------|
| Default | Track filled to current year, thumb at position | Page load (latest year) |
| Hover | Thumb grows from 16px to 20px, cursor: grab | Mouse enter thumb |
| Dragging | Thumb at 20px, cursor: grabbing, year updates live | Mouse down + drag |
| Focus-visible | 2px ring-[#00c8ff] around thumb | Keyboard focus |
| Disabled | Reduced opacity (0.4), no interaction | During playback animation |

### Play/Pause Button States

| State | Icon | Background | Trigger |
|-------|------|-----------|---------|
| Stopped (default) | Play triangle | rgba(0,200,255,0.1) | Initial state / after pause |
| Playing | Pause bars | rgba(0,200,255,0.2) | After play click |
| Hover | Same icon | rgba(0,200,255,0.2) | Mouse enter |
| Focus-visible | Same icon + 2px ring-[#00c8ff] | Same | Keyboard focus |
| At end | Play triangle (auto-pause) | rgba(0,200,255,0.1) | Playback reaches 2024 |

### Speed Badge States

| State | Display | Background | Trigger |
|-------|---------|-----------|---------|
| 1x (default) | "1x" | rgba(255,255,255,0.08) | Initial |
| 2x | "2x" | rgba(255,255,255,0.08) | 1 click from 1x |
| 0.5x | "0.5x" | rgba(255,255,255,0.08) | 1 click from 2x |
| Hover | Current label + brighter bg | rgba(255,255,255,0.15) | Mouse enter |

Speed cycle order: 1x -> 2x -> 0.5x -> 1x (loops)

### Line Chart States

| State | Behavior |
|-------|----------|
| Single country | 5 lines: MIND composite (2px solid) + 4 dimension lines (1px dashed, 50% opacity). Legend shows all 5 toggleable |
| Multi-country | 1 line per country using COMPARISON_COLORS (2px solid, MIND composite only). Legend shows country names |
| Hover on line | Crosshair tooltip showing year + score for all visible series |
| Empty (no data) | Gap in line series. Tooltip shows "No data" for that year |
| Loading | Skeleton pulse in chart container |

### Year Controls Bar (shared)

| State | Visibility | Content |
|-------|-----------|---------|
| Latest year active | Hidden | Not rendered |
| Historical year active | Visible, pinned above chart/map content | "Viewing: {year}" label + "Back to latest" link |
| Tab switch (Map/Country) | Persists across tabs when year != latest | Same controls shared |

## Animation & Transition Contract

| Trigger | Animation | Duration | Easing |
|---------|-----------|----------|--------|
| Year slider drag | Line chart vertical marker moves, all charts update | 0ms (immediate) | none |
| Playback frame advance | Year slider thumb moves, charts update, map recolors | Per frame: 1000ms (1x), 500ms (2x), 2000ms (0.5x) | ease-in-out (CSS transition on thumb position) |
| Playback start | Play icon transitions to pause icon | 150ms | ease-out |
| Playback stop | Pause icon transitions to play icon | 150ms | ease-out |
| Year bar show (year != latest) | Slide down from above chart content | 200ms | ease-out |
| Year bar hide (back to latest) | Slide up and hide | 200ms | ease-in |
| Line chart series render | ECharts default animationDuration for initial draw | 500ms | ECharts cubicOut |
| Line chart series update (year or country change) | ECharts animationDurationUpdate | 300ms | ECharts cubicInOut |
| Map recolor on year change | ECharts visualMap transition | 500ms | ECharts default |
| Dimension line toggle on/off | Line fades in/out | 300ms | ease-in-out |

### Line Chart Styling (Claude's Discretion)

- **MIND composite line:** 2px solid, with 6px circle markers at each year point
- **Dimension lines (single country):** 1.5px dashed, no markers, 60% opacity. Shown by default, toggleable via legend click
- **Multi-country overlay lines:** 2px solid, 4px circle markers, 100% opacity
- **Area fill:** None. Clean line-only aesthetic matching the dark theme
- **Animation easing for playback:** `ease-in-out` (CSS) for slider thumb movement. ECharts chart transitions use default `cubicInOut`
- **Tooltip:** Dark background (#0f0f18) with 1px border (#1e1e30), rounded-lg. Shows vertical crosshair line. Lists all visible series with color dot + label + score

## Data Flow Contract

### Year State Propagation

```
User action (slider drag / playback tick)
  --> state.setYear(year)
    --> subscribers notified
      --> timeline.ts: update line chart vertical marker
      --> init.ts: fetch getHistoricalSlimPayload(year), update radar + bar + binding constraint
      --> map.ts: fetch getHistoricalSlimPayload(year), re-render makeMapOption(data, dimension)
      --> url-state.ts: encode year= param in URL
```

### Historical Data Loading Strategy

- Historical JSON (`mind-scores-historical.json`, 318KB) injected as `<script type="application/json">` at build time (same pattern as current `indicator-data`)
- Parsed once on first year interaction, cached in memory
- No runtime fetch required -- all data is build-time embedded
- `getHistoricalSlimPayload(historicalData, year)` extracts SlimCountry[] for a given year
