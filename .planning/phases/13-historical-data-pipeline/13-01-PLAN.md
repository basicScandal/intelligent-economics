<objective>
Extend the World Bank data fetch pipeline to retrieve 10 years of historical data (2014-2024) for all 16 indicators across 217 countries. Produce a new `mind-scores-historical.json` with year-indexed MIND scores. The existing single-year `mind-scores.json` must remain unchanged so the current dashboard continues to work identically.
</objective>

<tasks>

## Task 1: Extend fetch script for historical data

**Files to modify:** `scripts/fetch-world-bank.ts`
**Files to create:** `scripts/fetch-world-bank-historical.ts`

Create a new fetch script `scripts/fetch-world-bank-historical.ts` that:

1. Reuses the existing INDICATORS mapping and helper functions from the original script
2. For each indicator, fetches data with `date=2014:2024` parameter (10-year range) instead of `mrnev=1`
3. Uses pagination (`per_page=1500`, paginate if more) to get all country-year combinations
4. Groups results by country ISO3 code AND year
5. Rate limits: 2.5s delay between indicator requests (same as original)
6. Handles missing data gracefully (some countries won't have data for all years)

Output structure for `src/data/mind-scores-historical.json`:
```json
{
  "metadata": {
    "generatedAt": "...",
    "yearRange": [2014, 2024],
    "indicatorCount": 16,
    "countryCount": 217,
    "version": "1.0.0"
  },
  "years": {
    "2014": {
      "countries": {
        "USA": { "m": 85.2, "i": 91.3, "n": 72.1, "d": 68.4, "mind": 78 },
        ...
      }
    },
    "2015": { ... },
    ...
    "2024": { ... }
  }
}
```

Each country-year entry stores only the 4 dimension scores (m/i/n/d) and composite MIND score — no raw indicator data (keeps payload manageable).

Add npm script: `"fetch-data-historical": "tsx scripts/fetch-world-bank-historical.ts"`

## Task 2: Shared data types for year-indexed scores

**Files to create:** `src/lib/historical-data.ts`

Create a shared types/helpers module:

```typescript
export interface YearScores {
  m: number | null;
  i: number | null;
  n: number | null;
  d: number | null;
  mind: number | null;
}

export interface HistoricalData {
  metadata: {
    generatedAt: string;
    yearRange: [number, number];
    indicatorCount: number;
    countryCount: number;
    version: string;
  };
  years: Record<string, { countries: Record<string, YearScores> }>;
}

/** Get available years from historical data */
export function getAvailableYears(data: HistoricalData): number[];

/** Get scores for a country across all years */
export function getCountryTimeSeries(data: HistoricalData, iso3: string): { year: number; scores: YearScores }[];

/** Get all countries for a given year */
export function getYearSnapshot(data: HistoricalData, year: number): Record<string, YearScores>;
```
