**Structure:**

# Discord Server Setup Guide

## Overview
Step-by-step guide to create and configure the Intelligent Economics Discord server.

## Step 1: Create the Server
- Go to discord.com, click "+" to create a new server
- Choose "Create My Own" -> "For a community"
- Server name: "Intelligent Economics"
- Upload server icon (use the site's network logo or similar)

## Step 2: Create Channels
Delete the default channels and create these:

**Information:**
- #welcome (read-only, rules and getting started)
- #announcements (read-only, updates from the team)

**Community:**
- #introductions (new members introduce themselves)
- #general (open discussion)
- #zone-zero-ideas (simulator experiments and proposals)

**Role-based:**
- #economists
- #engineers
- #designers
- #organizers
- #researchers
- #policymakers

## Step 3: Set Up Roles
Create these roles (matching the volunteer form role cards):
- Economist / Theorist (color: #00ff88)
- Engineer / Builder (color: #7b4bff)
- Designer / Communicator (color: #00c8ff)
- Community Organizer (color: #ffb400)
- Researcher / Academic (color: #ff6482)
- Policymaker / Advocate (color: #00c8ff)

Members can self-assign roles via reaction roles or a bot (suggest Carl-bot for free tier).

## Step 4: Pin "Start Here" Messages
In each role channel, pin a welcome message with:
- What this role contributes to the movement
- First suggested action (matching Email 3 micro-tasks)
- Link to relevant site section

## Step 5: Generate Invite Link
- Server Settings -> Invites -> Create invite
- Set to "Never expire"
- Copy the link
- Replace [DISCORD_INVITE_LINK] in:
  1. src/components/VolunteerForm.astro (success screen)
  2. docs/emails/01-welcome.md
  3. docs/emails/03-first-micro-task.md

## Step 6: Community Settings
- Enable Community Features (Server Settings -> Community)
- Set #welcome as rules channel
- Set #announcements as community updates channel
- Enable "Membership Screening" with a simple welcome gate

## Moderation
- Enable AutoMod for spam protection
- Set slow mode on #general (30 seconds)
- Consider adding MEE6 or Dyno bot for moderation at scale

This is a documentation-only artifact. The user follows these steps manually in Discord's web/desktop interface.
